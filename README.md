## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- Yarn or npm
- Docker (optional)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/VIDGENCRAFT/Vidgencraft.git
cd vidgencraft
```

2. Install dependencies:
```bash
yarn install
```

3. Set up environment variables:
```bash
cp .env.example .env
```

4. Start development server:
```bash
yarn dev
```
